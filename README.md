# Aprendiendo JavaScript Moderno

Repositorio de ejercicios para aprender JavaScript moderno como parte de un plan de estudio de desarrollo backend.

## Temas

- Arrow functions y template literals
- Destructuring de objetos y arrays
- Módulos: import / export
- Promesas y async/await
- fetch para consumir APIs externas
- Métodos de array: map, filter, reduce

## Meta

Construir un script que consuma una API pública y muestre datos. Base para el siguiente paso: Node.js + Express.
